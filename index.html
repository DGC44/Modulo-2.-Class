FUNDAMENTOS DEL DESARROLLO WEB

Display

La propiedad display en CSS controla cómo se muestra un elemento en una página web. Todos los elementos en HTML por defecto son rectangulares ya que internamente el navegador dibuja un rectángulo.
El comportamiento de todas las cajas proviene de la propiedad display de CSS. Esta propiedad especifica si un elemento es tratado como block o inline y el diseño usado por sus hijos, como grid (Cuadricula) o flex (Flexible).

Model-Box

Todo en CSS tiene una caja alrededor, y comprender estas cajas es clave para poder crear diseños con CSS o para alinear elementos con otros elementos.
Está formado por: contenido, relleno, bordes y márgenes.

Pilares de CSS

La Especificidad determina qué estilos se aplican a un elemento, en función de la especificidad de la regla CSS. Sistema de puntuación que asigna prioridad a los distintos tipos de selectores CSS.
La herencia está relacionada en cómo los elementos del etiquetado de HTML heredan propiedades de sus elementos padres y los transmiten a sus hijos.
La cascada resuelve los conflictos que se producen cuando varias declaraciones afectan a un elemento determinado, estas se anulan o no.

Flexbox

El Módulo de Caja Flexible, comúnmente llamado flexbox, fue diseñado como un modelo unidimensional de layout, y como un método que pueda ayudar a distribuir el espacio entre los ítems de una interfaz y mejorar las capacidades de alineación.
Cuando trabajamos con flexbox necesitamos pensar en términos de dos ejes: eje principal y eje cruzado. El eje principal está definido por la propiedad flex-direction, y el eje cruzado es perpendicular a este.

Flexbox Repaso

Flexbox es un método de diseño de página unidimensional para compaginar elementos en filas o columnas. Los elementos de contenido se ensanchan para rellenar el espacio adicional y se encogen para caber en espacios más pequeños.
Por ello, en esta clase se practico la creacion de una pagina web sobre SuperHeroes de Marvel. En donde se mostro con fichas para practicar distintas funciones de Flexbox.

Posicionamiento

Es un módulo de CSS que define cómo posicionar elementos absoluta y relativamente en la página, en relación con el flujo normal del documento y con su elemento padre.
Para definir la posición de un elemento, se utiliza la propiedad position y se le asigna uno de los siguientes valores: Static, Relative, Absolute, Fixed, Sticky. Para definir la ubicación final de los elementos posicionados, se utilizan las propiedades top, right, bottom y left.

Diseño Responsivo

En la actualidad, el uso de todo tipo de dispositivos móviles se ha disparado, no sólo de «smartphones», sino también de tablets, «smartwatchs», lectores de ebooks y múltiples tipos de dispositivos con capacidad de conexión a Internet. Cada vez es más frecuente acceder a Internet con diferentes tipos de dispositivos, que a su vez tienen diferentes pantallas y resoluciones, con distintos tamaños y formas.
